About this report:

This is a simple HTML report generated from your Playwright test results. It provides a clean and straightforward view of your test outcomes without requiring any special plugins or Java dependencies.

Running tests locally:

To run tests and generate reports locally, use the following commands:

# Run tests
npm test

# Generate HTML report
node utils/generateSimpleReport.js

# Open the report in your browser
open reports/index.html  # On macOS
# or
start reports/index.html  # On Windows
