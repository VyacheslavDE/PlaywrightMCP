This page provides access to the latest test reports for the Playwright MCP project.

View GitHub Actions

Instructions for viewing the report:

  1. Go to the latest successful workflow run
  2. Download the "allure-report" artifact
  3. Extract the ZIP file
  4. Open the "index.html" file in your browser

Running tests locally:

To run tests and generate reports locally, use the following commands:

# Run tests with Allure reporter
npm run test:allure

# Generate Allure report
npm run allure:generate

# Serve Allure report (requires Allure CLI)
npm run allure:serve
